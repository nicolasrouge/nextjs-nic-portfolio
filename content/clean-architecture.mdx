## Kestrel
<Image
  alt={`kestrel`}
  src={`/images/netcore/kestrel.png`}
  width={1600}
  height={840}
/>

IIS can typically be used as a reverse proxy that forwards requests to Kestrel. 
This can provide benefits such as load balancing, 
integration with IIS features, 
enhanced security, 
and compatibility with legacy applications.
The application code itself is typically hosted on a separate server or servers, and it communicates with the web server via HTTP or other protocols.
